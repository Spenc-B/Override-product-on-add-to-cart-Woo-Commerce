# Override-product-on-add-to-cart-Woo-Commerce

When a product is added to cart change the product type from lottery or whatever is stated to normal when added to cart and in checkout.

This will work for all products of type lottery in this case

For testing the product type has been placed under the image - wc-block-components-order-summary-item__image

Due to my case, I'll need to adjust the email template for a normal product as i don't sell these I can get away with it.
